

# Micro Services
Microservice to support the Avaition Tracking application

## Building all Services
Maven will automatically build all services, with a target version of Java 11, when run in this location

ex:  `mvn install`

### Building Services for Java 1.8 (8) targets.  
To build services targeted to Java 8, use profile `Java8` during build.

ex: `mvn install -P Java8`
